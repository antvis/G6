## 问题

说明：3.2.x 版本不会出现这个问题。

## 解决方案
1. 添加 babel-plugin-lodash 插件，此插件会自动优化 lodash 的引用方式。
2. 设置 @rollup/plugin-node-resolve 插件 browser 属性为 true，修复 G 当前的问题。
