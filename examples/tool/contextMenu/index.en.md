Using contextMenu in G6.

## Usage
The demo below show how to use contextMenu on a graph. Menu's style can be defined by the CSS:

```
/* Define the CSS with the id of your menu */
  #contextMenu {
    position: absolute;
    /* ... Other styles */
  }
  #contextMenu li {
    cursor: pointer;
		/* ... Other styles */
  }
  #contextMenu li:hover {
    color: #aaa;
    /* ... Other styles */
  }
`);
```
