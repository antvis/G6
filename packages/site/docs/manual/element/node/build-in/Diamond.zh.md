## 概述

菱形是一个四边相等的几何形状，具有对角线对称性。

适用场景：

- 用于表示决策节点、条件判断或关键路径。

- 适合表示流程图中的判断节点或关键步骤。

- 常用于流程图、决策树、网络图等。
