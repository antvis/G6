## 概述

椭圆是一个长轴和短轴不对称的几何形状。

适用场景：

- 用于表示动态节点、非对称关系或特殊形状。

- 适合表示流程图、网络图或拓扑图。

- 常用于流程图、网络图、拓扑图等。
