## 概述

矩形组合以直角边界组织内容，支持严格的层级结构。

适用场景：

- **系统架构图**：如系统架构里面的服务分层，以及每层服务里面的细分等。
- **地理区域划分**：如城市包含多个区域，矩形组合能直观展示行政边界或功能分区。
