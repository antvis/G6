## 概述

圆形组合以圆形边界包裹子节点或子组合，适合表示平等或非层级化的群组关系。

适用场景：

- 适合表示无明确层级关系的节点群组，圆形组合能体现成员的平等性，如社交网络中的用户群体、分散式团队结构（突出协作性）。
