支持为图画布设置一个背景图片，让画布更有层次感、叙事性。

<embed src="@/common/api/plugins/background.md"></embed>

## 配置项

### <Badge type="success">Required</Badge> type

> _`background` \| string_

此插件已内置，你可以通过 `type: 'background'` 来使用它。
