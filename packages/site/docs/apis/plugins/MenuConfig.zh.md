### handleMenuClick

**类型**：`(target: Item, item: MenuItem) => void`

**默认值**：`undefined`

点击菜单项的回调函数

### itemTypes

**类型**：`string[]`

**默认值**：`['node', 'edge', 'canvas']`

添加菜单项的对象类型

### liHoverStyle

**类型**：`CSSProperties`

**默认值**：

```json
{
  "color": "white",
  "background-color": "#227EFF",
};
```

菜单项 hover 状态的样式

### onHide

**类型**：`() => void`

**默认值**：`undefined`

菜单隐藏时的回调函数

### trigger

**类型**：`'click' | 'contextmenu'`

**默认值**：`'click'`

触发菜单的事件类型
