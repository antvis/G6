用户输入数据中的 `nodes` 部分内容。

下面是一个数据项实例：

```json
{
  "id": "node-1",
  "data": {}
}
```

## id <Badge type="error">必须</Badge>

节点的唯一 ID，节点创建后，ID 不可被修改。

**类型**：`string | number`

## data <Badge type="error">必须</Badge>

节点数据
