## DonutShapesStyle

**类型**：`DonutShapesStyle`

<details>

<summary style="color: #873bf4; cursor: pointer">DonutShapesStyle</summary>

```typescript
type DonutShapesStyle = CircleStyleProps & {
  /**
   * 甜甜圈的内径
   */
  innerRadius?: number;
  /**
   * 甜甜圈配置
   */
  donuts: Round[];
};
```

其中，相关的图形配置参考 [Circle 图形样式](../shape/CircleStyleProps.zh.md)。

</details>

默认值：`object`

<details>

<summary style="color: #873bf4; cursor: pointer">object</summary>

```json
{
  "innerRadius": 0.5,
  "donuts": [],
}
```

**类型**：`Round`

<details>

<summary style="color: #873bf4; cursor: pointer">Round</summary>

```typescript
type Round = {
  /**

  value?: number;

  color?: string;
  /**
   * <zh/> 其他圆弧(path)样式配置

  [key: string]: any;
```
