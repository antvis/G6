## delegateStyle

**类型**：

```ts
type delegateStyle = {
  fill?: string;
  stroke?: string;
  lineWidth?: number;
  lineDash?: number[];
  opacity?: number;
};
```

**默认值**：

```json
{
  "fill": "#F3F9FF",
  "fillOpacity": 0.5,
  "stroke": "#1890FF",
  "strokeOpacity": 0.9,
  "lineDash": [5, 5]
}
```

**是否必须**：false

**说明**：临时矩形的样式
